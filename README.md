# Study Dashboard — Ebbinghaus

Плагин для Obsidian, который хранит изученные темы и их материалы, показывает очередь повторений и календарь. Интервалы повторения по умолчанию: **1, 3, 7, 14, 30, 60, 120 дней**. Их можно изменить в настройках.

## Быстрая установка

1. Создайте папку `<ваше-хранилище>/.obsidian/plugins/study-dashboard-ebbinghaus/`.
2. Скопируйте в неё только `main.js`, `manifest.json` и `styles.css` из этой папки.
3. В Obsidian откройте **Settings → Community plugins**, включите плагин и перезагрузите Obsidian при необходимости.

## Сборка из исходников

После изменения `main.ts` выполните `npm install` и `npm run build`. Получившийся `main.js` замените в папке плагина. Для разработки используйте `npm run dev`: сборщик будет автоматически обновлять `main.js` после каждого сохранения файла.

## Использование

* Нажмите значок выпускной шапочки в боковой панели или выполните команду **Open study dashboard**.
* Нажмите **«Во весь экран»** или выполните команду **Open study dashboard in main workspace**, чтобы открыть дашборд отдельной вкладкой в основной рабочей области.
* Выберите **Добавить тему**, укажите дисциплину, курс, преподавателя, семестр, тип обучения и ссылки/заметки на материалы. Каждая строка материалов — отдельная ссылка или путь к заметке.
* Для темы автоматически создаётся Markdown-заметка в `Учёба/Темы` с YAML-свойствами и тегами: дисциплина, курс, преподаватель, семестр, тип и статус. Папку можно изменить в настройках плагина.
* В разделе «Сегодня» нажимайте **Повторить**. Плагин отмечает очередной блок повторения как выполненный.
* В «Календаре» при добавлении темы сразу появляются все блоки повторений. Пройденные блоки перечёркиваются.
* Во вкладке «Экзамен» выберите темы, которые хотите повторить за одну сессию; во «Прогрессе» — статистика и сводка по дисциплинам.

Плагин не требует внешнего Calendar-плагина: календарь встроен в дашборд, а напоминания хранятся локально в данных плагина.
